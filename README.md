# KeenKeeper

🔗 Live Website: https://keenkeeper-lake.vercel.app/

## 📌 Project Overview

KeenKeeper is a friendship relationship management web application where users can keep track of interactions with friends through calls, texts, and video calls. The app helps users maintain meaningful relationships by organizing communication history and interaction analytics in a clean and user-friendly interface.

---

## 🚀 Technologies Used

| Technology   | Purpose                         |
| ------------ | ------------------------------- |
| React.js     | Build the user interface        |
| React Router | Handle page navigation          |
| Tailwind CSS | Styling and responsiveness      |
| Chart.js     | Visualize interaction analytics |

---

## ✨ Key Features

### 1. Timeline Interaction Tracking

Users can log interactions such as calls, texts, and video calls with friends and view them in a timeline.

### 2. Smart Filtering & Searching

The timeline supports filtering by interaction type and searching by friend name or interaction category.

### 3. Analytics Dashboard

Interactive doughnut charts display communication statistics including total calls, texts, and video interactions.

---

## 📱 Responsive Design

The application is fully responsive and optimized for desktop, tablet, and mobile devices using Tailwind CSS.
